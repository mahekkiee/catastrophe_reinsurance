📊 Project Overview

🎯 Problem

Indian insurers need data-driven reinsurance triggers to balance catastrophe protection with cost.

💡 Solution

Markov regime-switching + Compound Poisson model using real IRDAI data (2014-2024).

📈 Impact

Optimal trigger: 800 Crores. Saves 3-5% in costs while maintaining 90% protection.

Key Finding: Catastrophic years (2015, 2018, 2021) show 3.7x more claims than normal years.
